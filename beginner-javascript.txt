Working with Arrays

    Find the Largest Number: Write a function that takes an array of numbers and returns the largest number in the array.
    Find the Smallest Number: Write a function that takes an array of numbers and returns the smallest number in the array.
    Get the Sum of Numbers: Write a function that takes an array of numbers and returns the sum of all the numbers in the array.
    Get the Average: Write a function that takes an array of numbers and returns the average.
    Filter Even Numbers: Write a function that takes an array of numbers and returns a new array containing only the even numbers.
    Filter Odd Numbers: Write a function that takes an array of numbers and returns a new array containing only the odd numbers.
    Reverse an Array: Write a function that takes an array and returns a new array with its elements in reverse order.
    Find Duplicates: Write a function that takes an array and returns an array of all the duplicate elements.
    Remove Duplicates: Write a function that takes an array and returns a new array with all duplicates removed.
    Find the Index of an Element: Write a function that takes an array and an element, and returns the index of that element (or -1 if not found).
    Check if an Array is Sorted: Write a function that takes an array and returns true if it’s sorted in ascending order, and false otherwise.
    Flatten a Nested Array: Write a function that takes a nested array and returns a flattened version of the array.
    Rotate an Array: Write a function that takes an array and a number, and rotates the array to the right by that number of positions.
    Get Unique Elements: Write a function that takes two arrays and returns a new array containing only the elements that are unique to each array.
    Find Common Elements: Write a function that takes two arrays and returns a new array containing only the elements found in both arrays.

Working with Strings

    Find the Longest Word: Write a function that takes a sentence (string) and returns the longest word.
    Find the Shortest Word: Write a function that takes a sentence (string) and returns the shortest word.
    Count Words: Write a function that takes a string and returns the number of words in it.
    Reverse a String: Write a function that takes a string and returns the reversed version of it.
    Check for Palindrome: Write a function that takes a string and returns true if it’s a palindrome, and false otherwise.

    Capitalize Words: Write a function that takes a string and returns a new string where the first letter of each word is capitalized.

    Convert to Lowercase: Write a function that takes a string and returns the string in all lowercase letters.
    Remove Whitespace: Write a function that takes a string and returns the string with all whitespace removed.
    Count a Specific Character: Write a function that takes a string and a character, and returns the number of times that character appears in the string.

    Get Substring: Write a function that takes a string, a starting index, and an ending index, and returns the substring within that range.

    Check if a String Contains Another String: Write a function that takes two strings and returns true if the first string contains the second string, and false otherwise.
